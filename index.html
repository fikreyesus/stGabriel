How to set up Google Sheets sync
1 Open Google Sheets, create a new spreadsheet. Add three tabs named exactly: Families, Payments, Config
2 Click Extensions → Apps Script. Delete any existing code. Paste the full code from the google_apps_script.js file.
3 Click Deploy → New deployment → Type: Web app → Execute as: Me → Who has access: Anyone → Deploy
4 Copy the Web App URL (ends in /exec), paste it above, click Save URL then Test Connection
Common reasons sync fails:
• URL is wrong or expired — re-deploy and copy a fresh URL
• Access is set to "Only me" — must be set to Anyone
• Script was saved but never deployed — you must click Deploy
• Sheet tabs are named wrong — must be Families and Payments exactly
• Apps Script needs authorization — click Review Permissions when prompted
