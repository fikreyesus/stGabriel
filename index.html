Google Sheets Backend Setup
  1. Open Google Sheets and create a new spreadsheet named St Gabriel Church DB.
  2. Create three sheets (tabs): Families, Payments, Config.
  3. In the Families tab, add headers in row 1:
    id | husband | wife | address | children | createdAt
  4. In the Payments tab, add headers in row 1:
    id | familyId | date | type | periodFrom | periodTo | amount | method | timestamp
  5. In your spreadsheet, click Extensions → Apps Script.
  6. Delete any existing code and paste the Apps Script code from the google_apps_script.js file.
  7. Click Deploy → New deployment, choose Web app, set "Who has access" to Anyone, click Deploy.
  8. Copy the Web App URL and paste it below.

Data is stored in your private Google Sheet. This app runs entirely in your browser.
